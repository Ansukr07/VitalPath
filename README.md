# VitalPath

VitalPath is a modern, high-fidelity patient intake and clinical decision-support platform designed for modern healthcare environments. It streamlines the connection between patients and clinicians through intelligent triage, report analysis, and data-driven suggestions—always keeping the doctor in the loop.

## 🚀 Key Features

- **Smart Triage Engine**: A rule-based assessment system that instantly prioritizes patients by severity.
- **Doctor Decision Support**: Clinicians receive evidence-based suggestions that are fully reviewable and overridable.
- **Patient Dashboard**: Simplified intake process with symptom tracking and medical report uploads.
- **Premium Design**: A high-performance, glassmorphic UI built with a clean charcoal/slate aesthetic.
- **Interactive Pricing**: Dynamic pricing plans powered by `framer-motion` for a premium user experience.

## 🛠 Tech Stack

### Frontend
- **React 19** with **Vite** for ultra-fast development.
- **Framer Motion** for sleek, high-fidelity animations.
- **Vanilla CSS** for precise, custom design control.
- **React Router 7** for seamless navigation.

### Backend
- **Node.js** & **Express** for a robust API layer.
- **MongoDB** for secure and scalable clinical data storage.
- **Axios** for unified API communication.

## 📦 Installation & Setup

### 1. Clone the Repository
```bash
git clone https://github.com/Ansukr07/VitalPath.git
cd VitalPath
```

### 2. Backend Setup
```bash
cd backend
npm install
# Create a .env file based on .env.example
npm start
```

### 3. Frontend Setup
```bash
cd ../frontend
npm install
npm run dev
```

## ⚖️ Disclaimer

**VitalPath is a decision-support tool, not a medical device.**  
It is designed to assist healthcare professionals in making informed decisions. It does not provide medical diagnoses or treatments. A licensed healthcare professional must always guide and verify every clinical decision made through the platform.
